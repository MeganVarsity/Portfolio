Christoffels Menu App

The Christoffels Menu App was developed to streamline restaurant menu management. The app allows users to display a day's menu across three categories—starters, main courses, and desserts. It provides the functionality to delete courses from the menu and add new dishes on a separate page. Users can input the dish name, assign it to a specific course, add the price, and provide a description. Additionally, the app calculates and displays the average price for each course, making it easier for restaurant staff to manage pricing. The total number of menu items is also shown at the top of the page for user convenience. I opted for a clean, sleek design that prioritizes ease of use while maintaining a professional aesthetic.
